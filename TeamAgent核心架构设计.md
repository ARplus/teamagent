TeamAgent核心架构设计

┌─────────────────────────────────────────────────────────┐
│                 TeamAgent Web Platform                  │
│  - 任务看板 UI                                           │
│  - 工作流可视化                                          │
│  - 人类决策界面（关键！）                                │
│  - 任务分配和状态追踪                                    │
└────────────┬─────────────────────────┬──────────────────┘
             │                         │
        WebSocket API             WebSocket API
             │                         │
    ┌────────▼─────────┐      ┌───────▼──────────┐
    │ Claude Code      │      │ Claude Code      │
    │ + TeamAgent Skill│      │ + TeamAgent Skill│
    │ (小敏的电脑)      │      │ (段段的电脑)      │
    └────────┬─────────┘      └───────┬──────────┘
             │                         │
         自动执行                    自动执行
             │                         │
    ┌────────▼─────────┐      ┌───────▼──────────┐
    │   需要决策时      │      │   需要决策时      │
    │ → 对话框通知人类  │      │ → 对话框通知人类  │
    │ → 在Web界面决策  │      │ → 在Web界面决策  │
    └──────────────────┘      └──────────────────┘



流程说明

本地 Claude Code (Skill)          云端 TeamAgent (Platform)
        ↓                                ↓
  WebSocket 长连接 ←────────────→  实时任务推送
        ↓                                ↓
   自动领取步骤                      任务工作流管理
        ↓                                ↓
  本地AI执行任务                    人类审核和决策
        ↓                                ↓
   提交结果到云端 ←────────────→  流转到下一步
